# 青芯驭码项目亮点文案

本文档基于当前项目“青芯驭码（aily blockly）”的现有功能、技术架构与产品定位整理，可直接用于海报、路演展示、项目申报书或答辩材料中的“项目亮点”部分。

---

## 一句话定位

青芯驭码是一套面向嵌入式开发、STEM 教学与端侧 AI 场景的桌面图形化编程 IDE，覆盖从“积木编程入门”到“代码开发、固件烧录、串口调试、AI 模型训练与部署”的完整链路。

---

## 海报版项目亮点

### 发展维度

- 项目已从基础可用阶段发展至“智能期”，当前版本已集成 AI 助手、MCP Agent 与端侧模型训练部署能力。
- 已形成较完整的产品能力闭环，覆盖 Blockly 图形化编程、Monaco 专业代码编辑、硬件烧录、串口监视、模型训练与部署等核心功能。
- 具备持续演进能力，后续可进一步扩展应用商店、模型商店、云空间协作与多端生态。
- 采用模块化架构设计，支持多板卡、多工具链与多 AI 接口接入，具备良好的扩展性与可维护性。

### 创新维度

- 创新性实现“积木编程 + 代码编程”双模同屏，打通从启蒙学习到专业开发的连续路径。
- 内置三层 AI 能力，包括代码补全、智能对话助手和基于 MCP 的 Agent 工具调用，形成 AI 原生 IDE 能力。
- 集成端侧 AI 模型训练、量化、部署与推理验证流程，降低 TinyML 和边缘 AI 的学习与使用门槛。
- 采用纯 JavaScript 的 ESP32 烧录方案，减少对 Python 环境的依赖，显著提升安装与使用便利性。

### 社会价值

- 面向中文用户打造本土化编程开发环境，降低学校、创客空间与培训机构使用嵌入式工具链的门槛。
- 支持离线编译、离线烧录与离线教学，适应机房、实验室及网络受限场景，提升教育可达性。
- 为嵌入式教学、硬件原型验证和国产化工具链建设提供可落地的开源解决方案。
- 推动 AI 与硬件教育普及，让更多学生和开发者接触端侧智能与智能硬件开发。

### 教育维度

- 支持从 K12 到高校、职教再到创客培训的多学段教学，满足不同层次学习者需求。
- 通过积木到代码的平滑过渡，帮助学生建立计算思维、工程思维和跨学科实践能力。
- 将编程、硬件、AI、模型部署整合到统一平台中，适合开展 STEM/STEAM 一体化教学。
- 可作为课程实训、毕业设计、竞赛训练和创新项目孵化的平台型工具。

### 团队维度

- 项目已形成前端界面、Electron 桌面端、硬件工具链、AI 能力与文档体系协同推进的完整开发结构。
- 团队具备跨方向整合能力，能够同时覆盖图形化编程、代码编辑、嵌入式工具链与 AI 功能开发。
- 代码、配置、板卡支持与文档体系较完整，体现出较强的工程化能力与产品化意识。
- 具备开源共建和二次开发基础，便于后续吸引合作方、教育机构与开发者共同参与生态建设。

### 行业现状与项目优势

- 当前同类工具普遍存在积木与代码割裂、AI 能力外挂化、硬件支持单一、环境配置复杂等问题。
- 青芯驭码通过“积木 + 代码 + AI + 硬件 + 端侧模型部署”一体化设计，形成了明显的差异化优势。
- 相较传统开发工具，项目更适合中文教学场景、创客教育场景与低门槛嵌入式入门场景。
- 项目兼具开源属性、教育属性与产业延展空间，具有良好的推广价值和品牌化潜力。

## 答辩或路演可用的一段话

青芯驭码聚焦“编程教育、嵌入式开发与端侧 AI”融合场景，致力于打造一套中文原生、开箱即用、可离线运行的桌面图形化编程 IDE。项目以 Blockly 与 Monaco 双模协同为基础，集成硬件烧录、串口调试、AI 助手、MCP Agent 以及端侧模型训练部署能力，打通了从编程入门到智能硬件落地的完整链路。项目不仅具有较强的技术创新性和工程实现能力，也兼具教学价值、社会价值与生态扩展潜力。
